#----Custom Drawers----

Drawers for Textures:
[Texture] big or small texture depending on config
[TextureNoSO] big or small texture depending on config, no scale or offset
[SmallTexture] always small texture
[SmallTextureNoSO] always small texture, no scale or offset
[BigTexture] always big texture
[BigTextureNoSO] always big texture, no scale or offset
[Gradient] gradient editor
[TextTexture] texture field that creates a texture from text - has to be imrpoved
[MultiSlider] slider with two values (defined as vector, x,y are slider values z,w are min,max
[TextureArray] field for a texture array. automatically converts gifs/images to a texture array
	put --frameCountProp=<property name> to automatically fill a float property when creating a texture array
[Vector2] vector with only fields for xy
[Vector3] vector with only fields for xyz

#----Property Additions----

--extraOffset=<int>
	addes and extra x-offset to the property

--hover=<text>
	text that is shown when hovering above property

--altClick=action:value
	performs an action when holding alt down and clicking on property
	actions:
		url: opens url in browser
